¿Cómo funciona?

En el ámbito del Internet de las Cosas (IoT), se establece un sistema mediante sensores o dispositivos que se comunican con la nube a través de diversas formas de conectividad. Una vez que los datos alcanzan la nube, el software procede a procesarlos y determina la necesidad de realizar acciones específicas, como ajustar los propios sensores o dispositivos, sin requerir intervención del usuario o mediante el envío de alertas.

Elementos

Este sistema de IoT se compone de cuatro elementos fundamentales: sensores o dispositivos, conectividad, procesamiento de datos e interfaz de usuario. Los sensores recopilan datos del entorno, como los presentes en un teléfono inteligente con GPS, cámara y acelerómetro. La conectividad se establece mediante Wi-Fi, Bluetooth, satélites, redes LPWAN o conexión directa a Internet. Los datos recopilados son procesados en la nube, donde el software decide si es necesario actuar, y la interfaz de usuario posibilita la intervención humana cuando es requerida.
